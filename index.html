Os Principais Comandos para Criar um Site

Comando HTML

O HTML (HyperText Markup Language) é a linguagem de marcação padrão usada para criar páginas web. O comando básico para criar uma página HTML é:

<!DOCTYPE html>
                <html>
                <head>
                    <title>Título da Página</title>
                </head>
                <body>
                    <h1>Olá Mundo!</h1>
                    <p>Este é um parágrafo.</p>
                </body>
                </html>

Comando CSS

O CSS (Cascading Style Sheets) é usado para estilizar a aparência das páginas web. Um exemplo de código CSS para definir o estilo de um título e parágrafo é:

h1 {
    font-size: 30px;
    font-family: Arial, sans-serif;
    color: blue;
}
h2 {
    font-size: 20px;
    font-family: Arial, sans-serif;
    color: red;
}
p {
    font-size: 15px;
    font-family: 'Times New Roman', serif;
    color: black;
}

EXPLICAÇÃO: 1. HTML (index.html): header: Contém o título principal da página. main: Contém as seções principais do site com títulos secundários e parágrafos explicativos. pre, code: Usado para exibir código HTML e CSS com formatação.

CSS (styles.css): Define a aparência dos títulos (h1 e h2) e dos parágrafos (p). O pre e code são estilizados para exibir o código de forma legível. O header e o footer são estilizados para melhorar a aparência geral da página.
