Principais Comandos para Criar um Site

Comando HTML Básico

O HTML (HyperText Markup Language) é a espinha dorsal de qualquer site. Aqui estão alguns comandos básicos:

  • <!DOCTYPE html>: Declara o tipo de documento e a versão do HTML.
  • <html>: O elemento raiz que encapsula todo o conteúdo da página.
  • <head>: Contém meta-informações sobre o documento, como título e links para estilos.
  • <body>: Contém o conteúdo visível da página, como textos, imagens e links.

Exemplo de um documento HTML básico:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu Site</title>
</head>
<body>
    <h1>Bem-vindo ao meu site</h1>
    <p>Este é um parágrafo de exemplo.</p>
</body>
</html>

Comando CSS Básico

O CSS (Cascading Style Sheets) é utilizado para estilizar os elementos HTML. Aqui estão alguns comandos e propriedades básicos:

  • selector { property: value; }: Sintaxe básica de uma regra CSS.
  • body { background-color: lightblue; }: Define a cor de fundo da página.
  • h1 { color: blue; }: Define a cor do texto do título principal.

Exemplo de um arquivo CSS básico:

/* Estilo para o corpo do documento */
body {
    font-family: "Times New Roman", serif;
    color: black;
    margin: 20px;
}

/* Estilo para os títulos */
h1 {
    font-size: 30px;
    font-family: Arial, sans-serif;
    color: blue;
}

h2 {
    font-size: 20px;
    font-family: Arial, sans-serif;
    color: red;
}

Explicação dos Códigos

index.html

Estrutura HTML:

'header': Contém o título principal da página.

'main': Contém as seções principais do conteúdo.

'section': Define seções distintas com títulos e parágrafos.

'footer': Contém informações sobre direitos autorais e outras informações de rodapé.

'pre' 'code': Usado para mostrar o código com formatação preservada.

style.css

Estilos Básicos:

'body': Define a fonte padrão e a cor do texto.

'h1', 'h2': Estiliza os títulos com tamanho, fonte e cor específicas.

'p': Define a fonte e o tamanho dos parágrafos.

'code': Estilo para código dentro dos parágrafos.

'img': Garante que as imagens se ajustem ao tamanho da tela.

'ul': Define o estilo das listas.

'pre': Estilo para blocos de código.
